# Autonomous Warehouse Navigation with ROS 2 Nav2

## Overview
This project demonstrates an autonomous mobile robot navigating a warehouse
environment using ROS 2 Humble and the Nav2 stack.

The robot is deployed in a realistic warehouse simulation and is capable of:
- Localization using AMCL
- Global and local path planning
- Obstacle avoidance
- Goal-based autonomous navigation

---

## System Architecture
- ROS 2 Humble
- Nav2
- Gazebo (Warehouse World)
- TurtleBot3
- AMCL Localization
- Behavior Tree Navigator
